06 SSH 보안 설정
변경 전 (위험)
Port22
PermitRootLoginyes
변경 후 (안전)
Port20022
PermitRootLoginno
왜 바꿔야 하는가?
포트 22는 전 세계 해킹 봇의 기본 공격 대상이다. 포트를 바꾸는 것만으로도 자동화 공격 시도를 대폭 줄일 수 있다.
루트 계정은 시스템 전체 권한을 가지므로, 원격 로그인이 허용되면 침해 시 피해가 치명적이다.

리소스 수집
# CPU 사용률
CPU=$(top -bn1 | grep "Cpu"
  | awk '{print $2}')

# 메모리 사용률
MEM=$(free | grep Mem
  | awk '{printf "%.1f", $3/$2*100}')

# 디스크 사용률
DISK=$(df / | tail -1
  | awk '{print $5}' | tr -d '%')
로그 기록
# 타임스탬프 생성
TS=$(date '+%Y-%m-%d %H:%M:%S')

# 로그 파일에 추가
echo "[$TS] PID:$PID
  CPU:${CPU}%
  MEM:${MEM}%
  DISK_USED:${DISK}%"
  >> $LOG_FILE
